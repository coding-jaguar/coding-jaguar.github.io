Procedure

    1. Refer to the following diagrams to make connections in the simulation for Differential Amplifier.
    2. Connect the wires in the simulation as per the given diagram

    3. After connections are made, click on 'check' to check them.
    4. If unable to make correct connections, click on 'connect automatically'
    5. It is necessary to click on 'check' to set resistance values automatically and begin calculations
    6. Set various values of input voltages click on 'calculate' to fill observation table rows
    7. Use CMRR values from the table to calculate Noise%
