Procedure

  1. Refer to the following diagrams to make connections in the simulation for Inverting Amplifier.
  2. Refer to the following diagrams to make connections in the simulation for Non-inverting Amplifier.
  3. After connections are made, click on 'check' to check them.
  4. If unable to make correct connections, click on 'connect automatically'
  5. Set various values of input voltage, input frequency,R1 and R2, and click on 'calculate' to plot graph
  6. Do the same for all three configurations provided in the dropdown box.
